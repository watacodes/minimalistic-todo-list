# Project

Minimalistic Todo — Next.js + TypeScript + Tailwind + shadcn/ui

# Demo & Stack

- Stack: Next.js(App Router), TypeScript, Tailwind v4, shadcn/ui

# Features

- Add / Toggle / Delete
- Local persistence
- Keyboard entry support

# Future improvements

- Filters
- Counter for todo items
- Inline Edit

# Getting started

```
pnpm i (or npm/yarn)
pnpm dev
pnpm build && pnpm start
```
